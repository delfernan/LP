This project is maintained by delfernan

The main purpose of the LP Demo project is to serve as a reference for the development of Project Based Learning (PBL) methodology in which the programming language selected is C ++ and integrates QT libraries for developing graphical user interfaces.

It is intended that this project never ends, so every student contribution is sincerely appreciated.
